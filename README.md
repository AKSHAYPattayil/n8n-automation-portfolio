# n8n Automation Portfolio

A curated collection of production-style automation workflows built using **n8n**.
Each automation focuses on **clarity, scalability, and real-world use cases** .

---

## What This Repository Contains

- End-to-end workflow automations
- Schema-driven input handling
- AI-integrated pipelines
- Modular, reusable workflow design
- Clear documentation for each automation

Each folder contains:
- Exported n8n workflow (`workflow.json`)
- Architecture explanation
- Screenshots of the workflow canvas
- Setup instructions

---

## Design Principles

- Credentials are never hard-coded
- Workflows are modular and extensible
- Nodes are grouped and labeled for readability
- Failures are handled explicitly where relevant

---

## Tech Stack

- n8n
- JSON Schema
- LLM APIs
- Webhooks & Forms
